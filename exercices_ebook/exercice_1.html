Question, réponses exercice 1

Question 1
La déclaration DOCTYPE est-elle obligatoire ? Où doit-elle être placée ? Quelles informations contient-elle ?
Réponse 1 : La déclaration DOCTYPE est obligatoire, elle est la structure réfencielle, elle se place en tête du code devant la balise <html>Elle contient le nom de l'élément racine
Question 2
Que faut-il faire pour que le code HTML soit valide ?
Réponse 2 : Il faut respecter l'inclusion ou l'imbrications des éléments et les attributs propres à chacun d'entre eux et ensuite vérifier avec le validateur du W3C
Question 3
À quel emplacement est defini le jeu de caractères utilisé dans le document ?
Réponse 3 : le jeu de caractères est précisé dans un élément <meta> avec l'attribut charset inclus dans<head> <meta charset="UTF-8">
Question 4
Où est déclaré le nom de l'élément racine du document ? Quel est son rôle et son nom ?
Réponse 4 : Le nom de l'élément racine est donné dans DOCTYPE,il s'agit de <html>; c'est l'élément qui inclus tous les autres donc il est au sommet de la hièrarchie
Question 5
Quels sont les éléments enfants de l'élément HTML ?
Réponse 5 : <Les enfants (de premier niveau) sont <head> et <body>
Question 6
À quoi sert l'élement <head> ? À quoi sert-il ?
Réponse 6 : l'élément head est l'en-tête du document, il n'est pas vible dans le navigateur (sauf le tilte) : il conient BASE, LINK, META, SCRIPT, STYLE, TITLE. Seuls les éléments BASE et TITLE ne doivent être utilisés qu'une seule fois.
Question 7
Quel élement est obligatoire dans l'élément <head> ? À quoi sert-il ?
Réponse 7 : Seul l'élément TITLE est obligatoire. C'est le titre dans l'onglet du navigateur.
Question 8

Réponse 8 :<link rel="stylesheet" href="css/style.css">
